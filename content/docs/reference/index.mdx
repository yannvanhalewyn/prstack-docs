# Command Reference

This section provides detailed documentation for all PrStack commands and their options.

## Quick Reference

| Command | Purpose | Usage |
|---------|---------|-------|
| [`prstack`](./tui) | Launch Terminal UI | Interactive stack management |
| [`prstack sync`](./sync) | Sync stack with remote | Run frequently to keep stack in sync |
| [`prstack list`](./list) | View stack structure | Quick overview of your current stack |
| [`prstack create-prs`](./create-pds) | Create PRs | Batch create PRs for your stack |
| [`prstack feature-base`](./feature-base) | Manage Feature Bases| Create, add and remove feature base branches |
